# PushupMinutes

Android app prototype that earns minutes of social media usage through counted pushups.

## Status
This is an early scaffold with UI screens, data storage, and placeholders for camera pose detection and usage monitoring.

## Requirements
- Android SDK 34
- Kotlin
- Gradle (Android Studio recommended)

## Quick Start
- Open the project in Android Studio.
- Sync Gradle.
- Run on a device with Android 13+.

## Permissions
The app will request:
- Camera (pushup detection)
- Usage access (to monitor target apps)
- Notifications (time over message)

## Notes
- Camera pose detection and usage monitoring are scaffolded and need wiring.
- The arm illustration is a placeholder vector and can be replaced later.
